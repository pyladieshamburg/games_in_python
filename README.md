# Simple example of a game in python

### Tech requirements:
1. Python3 (3.5+)
2. Some editor =)

### Run locally:
1. ```python3 -m venv env```
2. ```source env/bin/activate```
3. ```pip install -r requirements.txt ```
4. ```python main```
